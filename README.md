# Goal Atomizer ⚛️

An AI-powered goal deconstruction tool. Break any ambitious goal into tiny, atomic micro-steps across four distinct phases (Foundation, Momentum, Execution, Mastery).

## Features
- **Multi-Model Support:** Integrates with Anthropic (Claude 3.5), OpenAI (GPT-4o), and Groq (Llama 3).
- **Persistence:** Progress and notes are saved to LocalStorage.
- **Markdown Export:** Download your generated plan as a professional `.md` file.
- **Adaptive UI:** Dark-themed, high-performance interface with custom SVG progress tracking.

## Tech Stack
- React
- Vite
- DM Sans & Syne Google Fonts

## How to use
1. Clone the repo.
2. Run `npm install`.
3. Run `npm run dev`.
4. Paste your API Key and enter a goal!
